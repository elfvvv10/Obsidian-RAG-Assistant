# Projects

Use this folder for track-specific work.

Recommended pattern:
- `Current Tracks/<Track Name>/` for active tracks
- `Track Ideas/` for loose concepts that are not active yet

Within an active track folder, keep:
- `track_context.md` for legacy markdown track context if you use it
- `session_notes/` for working-session notes
- `arrangements/` for structure ideas
- `sound_design/` for patch and texture work
- `exports/` for bounce references
