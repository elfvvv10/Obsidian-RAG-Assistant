## Sound Design Brainstorm

Quick Answer: Start with a simple, evolving bassline that builds tension before dropping.

Production Recipes:

1. Name: Classic Progressive House Shaker
   Groove / MIDI: Use a repeating low-mid pattern (e.g., E3-E4) for the first section, gradually adding notes in half steps to increase tension.
   Sound Design: A sawtooth or square wave with subtle distortion gives a hard-hitting, punchy feel. 
   How to Build It:
    1. Create a new synth and assign it a sawtooth or square wave.
    2. Use a low-mid frequency like E3-E4 as your starting pattern.
    3. Add subtle distortion to the synth for grit.
    4. In the first section, keep playing the repeating notes (E3-E4 in this example) until you build up tension by gradually adding half steps between them: E3, F♯3, G♯3, A3.
   Where to Use It: Use this drop-building pattern in the intro or breakdown sections.

2. Name: Funky Progressive House Groove
   Groove / MIDI: Design a catchy bassline with a mix of quarter notes and eighth notes for an upbeat feel.
   Sound Design: A plucky, distorted electric bass sound evokes a live instrument vibe.
   How to Build It:
    1. Use the piano roll or MIDI editor to create your funky groove pattern.
    2. Mix in-quarter and eighth-note patterns for interest.
    3. Try using D4-C5-B♯4-A♯4 as one option, with an eighth note on A♯4.
    4. Add a plucked electric bass sound from a sample library or synth plugin.
   Where to Use It: This versatile groove works well in the main drop, build sections, or even during transitions.

Optional Variations:
- Add a sub-bass kick to your bassline for an extra boost in low end presence.
- Experiment with different oscillator types like sine waves for a more melodic and smooth bass sound. 
- In the breakdown, try using a pad or string synth to add depth and emotion before transitioning back into the drop.
