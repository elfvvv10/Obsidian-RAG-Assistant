## Sound Design Brainstorm

Quick Answer: Progressive house basslines often use a combination of smooth, plucked notes and melodic synth lines to create an energetic yet groovy atmosphere.

Production Recipes:
1. Name: Plucky Bass Synth Combo
   Groove / MIDI: Create a simple 4/4 groove at 128 BPM using whole note values for the bassline on the downbeats with occasional eighth note plucks.
   Sound Design: Use a digital piano or acoustic guitar sample for the plucked bass, and layer it with a synth playing a repeating melodic line in parallel thirds or sixths above the bass. 
   How to Build It:
   1. Drag a sampled bass onto an empty MIDI track set to C3.
   2. Set the velocity curve to a simple shape, like a triangle wave, with no negative values. This will make the plucks sound consistent.
   3. Create a new MIDI track for the synth line above the bass. Quantize it to your project's tempo and turn off any swing or groove effects.
   4. Choose a bright, analog-sounding synthesizer preset and place it on this track. Program a simple melodic phrase using parallel thirds or sixths that moves with the bass but doesn't directly copy it note-for-note.
   5. Automate the volume of the synth so it gradually fades up with the bass to create a smooth transition into the drop.
   Where to Use It: This combination works well for the main build-up section, as the melodic synth adds tension while the plucky bass anchors the groove.

2. Name: Reese Bass and Analog Synth
   Groove / MIDI: Program a Reese-style bassline in 4/4 at 125-130 BPM using long notes with subtle movement on individual voices.
   Sound Design: Layer a supersaw or detuned saw oscillators for width, add some tube or transistor distortion to the signal after filtering, and use a filter cutoff envelope that moves with the bass. 
   How to Build It:
   1. Create a new MIDI track set to your target BPM and turn off any quantization effects.
   2. Program a Reese-style bassline using long notes and occasional minor second movement between voices for texture. Try using notes in the F# minor pentatonic scale (F#, A, C#, D#) for a common progression.
   3. Assign this to a supersaw oscillator with wide detune values between the unison voices. Adjust the width until it fills out the stereo field without sounding too artificial.
   4. Add some tube or transistor distortion to the signal after filtering (not on the individual oscillators). This will add warmth and analog grit without muddying up the sound.
   5. Automate a filter cutoff envelope that moves with the bassline, adding subtle changes in width over time. 
   Where to Use It: The Reese bass works well for the main build-up section or as a lead bass in more intense progressive house tracks.

Optional Variations:
1. Jungle-Inspired Bass and Synth Stabs
   Groove / MIDI: Program short, stabby notes in a syncopated 4/4 groove at around 165 BPM using whole notes and eighth notes for texture.
   Sound Design: Use sampled soul or jazz chords as your synth stabs, with occasional vocal samples layered on top. 
   How to Build It:
   1. Set up two new MIDI tracks one octave apart from each other. Quantize them both to a syncopated swing feel at 165 BPM.
   2. Drag in short, stabby notes using whole notes and eighth note values for texture. Experiment with minor chords or seventh chords for an urgent sound.
   3. For added energy, layer in some vocal samples from classic reggae or dub tracks. Pitch them down by a perfect fourth to make them feel more modern.
   Where to Use It: This combo works well in the build-up section of a peak-time progressive house track to give it some jungle-influenced energy and texture.

2. Building Tension with Risers, Downlifters and Impacts
   Sound Design: Program large, sweeping white noise sweeps for risers, drop down to silence for downlifters, and use pitched oscillators or samples for impact sounds.
   How to Build It:
   1. Create a new track for riser effects. Automate a large amount of white noise up in volume from negative dB levels up to around 0 dB over the course of a full bar. 
   2. For downlifters, set up another track and automate it so that it drops back down to silence after hitting its peak. You can also use samples of classic drum breaks for this effect.
   3. On another track, program impact sounds using pitched oscillators or sampled one-shots. Trigger them on the downbeat of each new section or build-up phrase for maximum impact.
   Where to Use It: This technique works well throughout the arrangement to help create a continuous sense of tension and release that builds towards euphoric peaks in your progressive house track.
